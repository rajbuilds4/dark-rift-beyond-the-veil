# Dark Rift II – Beyond The Veil

Dark Rift II is a story-driven action platformer I built for a hackathon.

The game takes place in a dark dimension called the Rift. After entering a mysterious portal, the player must survive enemy attacks, escape the Collapse, defeat powerful bosses, and eventually make a choice that changes the ending of the story.

This project was built using HTML5 Canvas and JavaScript. I focused on the game concept, story, gameplay mechanics, progression system, combat ideas, and endings while using AI-assisted development to speed up implementation.

## Features

* Melee combat system
* Dash and special burst abilities
* XP and level progression
* Multiple enemy types
* Boss fight with different phases
* Combo and scoring system
* Dynamic difficulty scaling
* Cinematic ending sequence
* Multiple endings based on player choice
* Custom audio and visual effects

## Controls

* A / D → Move
* W / Space → Jump
* Z → Attack
* X → Dash
* C → Burst Ability

## Why I Built This

I wanted to create a game that combined fast-paced combat with a story-driven experience. Instead of having a fixed ending, I wanted players to decide whether to return to peace or claim the power of the Rift for themselves.

## Hackathon

Dark Rift II was created as a hackathon project and won first place.
